### 年月日求星期

```java
class Solution {
    public String dayOfTheWeek(int day, int month, int year) {
        //参考蔡勒公式
        String[] res = {"Sunday", "Monday", "Tuesday", "Wednesday", "Thursday", "Friday", "Saturday"};
        if(month < 3){
            year--;
            month += 12;
        }
        int c = year/100;
        int y = year - c*100;
        int week = y + y / 4 + c / 4 - 2 * c + 26 * (month + 1) / 10 + day - 1;
        week = (week%7 + 7) % 7;
        return res[week];
    }
}
```
