# 前言

总结java中的访问权限控制,基础性的知识

## 访问权限一览

|           | 同一个类 | 同一个包 | 不同包中的子类 | 不同包中的非子类 |
| --------- | -------- | -------- | -------------- | ---------------- |
| public    | OK       | OK       | OK             | OK               |
| protected | OK       | OK       | OK             |                  |
| package   | OK       | OK       |                |                  |
| private   | OK       |          |                |                  |

## 关于private

除了做封装外，也常常用在单例模式

```java
public class Chicken {
    private  Chicken(){}
    static Chicken makeAChicken(){
        return new Chicken();
    }
}

//隐藏了构造器
Chicken c=Chicken.makeAChicken();
```
